## BottleNotes - MERN + Redux
#### Note taking and storing web app


#### Made using :
    1. MongoDB
    2. ExpressJS
    3. React + Redux
    4. jwt
    5. bcryptjs
    
#### IN config/default.json add following
    "mongoURI": "YOUR_MONGO_URI",
    "jwtSecret": "yoursecret",
